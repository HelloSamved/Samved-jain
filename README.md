# GitHub Pages Showcase

A sleek, modern web application to organize and showcase your GitHub Pages projects.

## ‚ú® Features

- **Project Management**: Easily add and remove links to your hosted websites.
- **Local Storage**: Your project list is saved locally in your browser, so it's there when you come back.
- **Premium Design**: Built with a "Glassmorphism" aesthetic, featuring smooth gradients and micro-animations.
- **Responsive**: Works perfectly on desktops, tablets, and mobile devices.
- **Interactive UI**: Powered by Framer Motion for delightful transitions.

## üìù How to Use

1. Click the **"+ Add Project"** button at the top right.
2. Enter a **Title** for your project (e.g., "My Portfolio").
3. Enter the **GitHub Pages URL** (e.g., `username.github.io/repo-name`).
4. Click **"Save Project"** to add it to your showcase.
5. Click **"Visit Site"** on any card to open the live project in a new tab.
